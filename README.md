# 나만의 취향 보물 아카이브 (MyTreasureArchive)

> **"좋아하는 순간들은 사라지지 않도록, 나만의 보물함에 차곡차곡 담아두어요."**

 **MyTreasureArchive는 내가 좋아하는 노래, 영화, 맛집을 저장하고  
한 줄 이유와 함께 기록하는 개인 취향 아카이브 프로젝트입니다.**

---

## 프로젝트 컨셉 (Design Concept)

단순한 CRUD를 넘어, 내가 좋아하는 것들을 한곳에 모아두는  
**나만의 취향 보물함**을 목표로 했습니다.

- **전체가 모이는 메인 페이지**  
  앱에 들어오면 모든 취향 데이터가 한눈에 보이는 리스트 형태로 구성했습니다.

- **랜덤 보물 카드**  
  메인 상단에는 정사각형 카드로 오늘의 랜덤 추천 항목을 띄워 보물함 감성을 더했습니다.

- **카테고리별 정리**  
  노래, 영화, 맛집을 각각 구분해 페이지별로 관리할 수 있도록 구성했습니다.

- **짧은 기록 중심**  
  상세 페이지에서는 간단한 정보와 함께, 내가 왜 좋아하는지 한 줄로 남길 수 있습니다.

- **개인용 아카이브**  
  댓글이나 커뮤니티 기능 없이, 혼자 쓰는 기록형 서비스로 설계했습니다.

## 주요 기능 (Key Features)

### 1. 메인 페이지 (Main List)
- 전체 취향 데이터(노래/영화/맛집)가 한 화면에 모여 있는 보물함 형태의 리스트를 제공합니다.
- 최신순 정렬로 최근 추가한 항목을 쉽게 확인할 수 있습니다.
- 상단 랜덤 카드에서 오늘의 보물을 하나 추천해줍니다.

---

### 2. 카테고리별 목록 (Category List)
- 노래, 영화, 맛집을 각각 분리한 페이지에서 관리할 수 있습니다.
- 카테고리별로 CRUD 기능을 수행할 수 있도록 구성했습니다.

---

### 3. 상세 페이지 (Detail View)
- 선택한 항목의 제목과 간단한 정보를 확인할 수 있습니다.
- 내가 이 항목을 좋아하는 이유를 한 줄 기록으로 남길 수 있습니다.

---

### 4. 등록 및 수정 (Register & Update)
- 새로운 보물을 추가하고, 기존 기록을 수정할 수 있습니다.
- 입력 항목은 단순하게 유지하여 부담 없이 기록할 수 있도록 했습니다.

---

### 5. 삭제 기능 (Delete)
- 더 이상 저장하지 않는 항목은 삭제하여 보물함을 정리할 수 있습니다.
